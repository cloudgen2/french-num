# french-num
A french number exercise written in Rust

## Update

### Version v0.2.9 
 * Update process_line regex

### Version v0.2.10
 * Added Pear, Cherry and WaterMelon

## Run the source code
```
cargo run
```

## Build release
```
cargo build --release
rm -rf ~/.local/bin/spanish-num
cp target/release/french-num ~/.local/bin/
```
